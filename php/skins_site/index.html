/skins_site folder

This folder holds the skins that apply to the whole site (i-e, not specific to a particular collection or type of collection).

This folder holds a set of default behavior include files for several functions. Each one can be overriden and customized. For example /skins_site/_xxx.php can be overridden by /skins_site/custom/_xxx.php.

This folder needs to be accessed through the web.
