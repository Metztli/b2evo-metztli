/skins folder

This folder holds the skins for the front-office. Each collection may use a different skin. It may actually even use different skins for web, tablet and mobile devices.

Each available skin can be found in its own sub-folder. You can install/create additional skins by copying/creating new subfolders into the /skins folder.

Subfolders starting with a `_` represent available skins which will remain hidden to the users. Those are typically used for RSS skins.

This folder needs to be accessed through the web.
